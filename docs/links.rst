Liens utiles
============

Unofficial Core Space App
    https://xinix.github.io/core-space/

    Une magnifique application qui recense tout les objets disponibles dans le jeu et
    toute ses extensions.

    Chaque objet contient un descriptif de son fonctionnement, caractéristiques, etc..

    Dans la configuration vous pouvez activer les extensions que vous utilisez pour
    afficher leurs objets spécifiques.

Aides de jeu de VortexZone
    https://vortexzone.com/core-space.html

    La communauté de VortexZone a créé une plaquette imprimable d'aides de jeu.
